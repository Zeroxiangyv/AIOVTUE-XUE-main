# 关于本站

这是专门用来记录我的生活的专属站点，记录生活中每一件难忘或者有意义的事情，分享教程之类的可以去主站点哦 [AIOVTUE](https://20030327.xyz)

另外，本站基于 [valaxy](https://github.com/YunYouJun/valaxy)和[sakura](https://github.com/WRXinYue/valaxy-theme-sakura), 你或许会问为什么底部没有熟悉的 `Valaxy驱动|主题Sakura`，这是因为我想把这个站点弄得简约一点，所以隐藏了，但是为了保护作者权益，在这里放一下地址。

## 联系方式

偌大的互联网，我想也不会有人没事联系我，毕竟这只是我分享生活的小树洞，不过要是真有人的话可以在下方留言评论，我会收到邮件，或者在主页联系站点图标选一个你喜欢的联系我。

## 友链
还有一点哦，本站支持交换友链的，只要是非免费二级域名的https站点都可以申请哈，加这个限制也主要是为了站点稳定，毕竟我也不想动不动就编辑，本着双向的原则，申请之前记得先添加本站的友链哦（我看他们都这样~虽然我并没有什么感觉，可能这样省事点？）好啦，有意向可以在这里也可以在友链页面评论哈，如果我能收到邮件（服务没出问题的话）我会很快添加并回复邮件的。格式在这里啦：
```
- name: #站点名称
  link: #站点链接
  avatar: #站长头像
  descr: #站点描述
  siteshot: #站点截图
```
